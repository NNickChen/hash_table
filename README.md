# hash_tab
This is the fastest contact searcher in the world!
You can search a contact contains more than 10000 indexes!
# How to use it
## Make a list (txt file) like:
---------------
Fire 119
Police 110
Accident 122
Ambulance 120
0
--------------
It must end with 0
## type filename
$ ./Hash_Table filename
or like
$ ./Hash_Table
Enter the filename: filename
